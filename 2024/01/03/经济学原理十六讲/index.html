    6. 固定成本：成本总额在一定时期和一定业务量范围内，不受业务量增减变动影响而能保持不变的成本
    7. GDP：某一既定时期内一个经济社会运用生产要素所产生的全部最终产品的市场价值。

    11. 消费者偏好的特点：完备性，可传递性，非饱和性
    12. 无差异曲线上任何一点的边际替代率等于它们的边际效用之比

第一章：

第一节

  1. 需求
  2. 需求的基本影响因素
  3. 需求定理
  4. 需求量
  • 答案：
    1. 需求：消费者在某一时期内在各种可能的价格水平上愿意而且能够购买的商品数量
    2. 需求的基本影响因素：1.商品自身价格， 2.相关商品价格 3.消费者的收入水平 4.消费者的口味或消费习惯 5.消费者对价格的预期 6.其他因素（天灾，天气，政策等）
    3. 需求定理：在其他条件不变的情况下，商品的需求量与价格之间通常呈反方向变动的关系，例外情况：炫耀性商品，极为抵挡的商品
    4. 需求量：（易与需求混淆）对一种商品的需求量是在一定价位下买者愿意并且能够购买的商品数量，需求量的变化，商品自身价格引起需求量的变化，在需求曲线中表现为点沿线的运动（需求的变化为线的平移）

第二节

  1. 供给
  2. 供给的影响因素
  3. 供给规律
  4. 供给量的变化与供给的变化
  • 答案：
    1. 供给：在一定时期内，在一定价格条件下企业愿意生产并销售某种商品的数量
    2. 供给的影响因素：商品本身价格
    3. 供给规律：在其他条件不变的情况下，供给量与商品价格呈现正相关，例外情况：名画，古玩，
    4. 供给量的变化与供给的变化：供给量的变化是指商品价格变化引起的商品供给数量的变动，在供给曲线上表示为点的移动，商品本身价格不变的情况下，其他因素引起的变化为供给的变动。

第三节

  1. 均衡价格与均衡数量
  • 答案：
    1. 均衡价格与均衡数量：市场达到均衡时的价格为均衡价格，和均衡数量

第四节

  1. 需求的价格弹性
  2. 需求价格弹性的影响因素
  3. 供给弹性的分类
  • 答案：
    1. 需求的价格弹性：需求量变动对于商品本身价格变动的敏感程度
    2. 需求价格弹性的影响因素：商品的可替代程度（正），占收入的比重（正），商品用途的多少（正），时间框架（正）
    3. 供给弹性的分类： > 1：富有弹性， < 1：缺乏弹性， = 1：单一弹性，= 0：完全无弹性， = ∞：无穷大弹性

第二章

第一节

  1. 消费者偏好
  2. 消费者偏好的特点
  3. 效用
  4. 总效用
  5. 边际效用
  6. 边际效用递减规律
  7. 需差异曲线
  8. 无差异曲线的特征
  9. 需算约束线
  • 答案：
    1. 消费者偏好：消费者根据自己的意愿，对可能消费的商品组合（商品束）进行的排序
    2. 消费者偏好的特点：完备性，可传递性，非饱和性
    3. 效用：消费者从商品和服务的消费中所得到的满足程度（个人心理感受）
    4. 总效用：消费中在一定时期内消费一定数量的商品或服务所得到的总满足程度或效用总量
    5. 边际效用：一定时期内消费中每增加或减少一单位商品或服务的消费时所得到的总效用的增加或减少量
    6. 边际效用递减规律：在一定时间内，在其他商品的消费数量保持不变的条件下，随着消费者对某种商品消费量的增加，消费者从该商品连续增加的每一消费单位中所得到的效用增量即边际效用是递减的
    7. 无差异曲线：满足消费者效用水平不变的情况下，两种商品消费数量的不同组合
    8. 无差异曲线的特征：1.斜率为负，2.离原点越远的无差异曲线代表的效用水平越高， 3.任意两条无差异曲线不能相交，4.无差异曲线的形状在多数情况下是凸向原点的
    9. 预算约束线：（预算线，消费者可能线和价格线）消费者的收入和商品的价格给定的条件下，消费者的全部收入所能购买到的两种商品的各种组合

第三章

第一讲

  1. MP,TP,AP
  2. 边际报酬递减规律
  • 答案：
    1. MP(边际产品)，TP(总产品)，AP(平均产品)：其曲线，三条曲线都是先升后降，MP最先达到最大值，其次AP达到最大值，最后TP达到最大值，在MP为0时，TP开始下降
    2. 边际报酬递减规律：技术水平不变的情况下，若生产组合中的其他要素投入固定不变，持续增加一种可变要素的数量，最初边际产量递增，当该要素增加到一定数量后，再增加其边际产量将逐渐减少。

第三节 企业的成本分析

  1. 成本
  2. 显性成本
  3. 隐性成本
  4. 机会成本
  5. 可变成本
  6. 固定成本
  7. 边际成本
  8. 边际成本函数和平均成本函数图形
  • 答案：
    1. 成本：企业支付给生产要素的报酬，即生产一定数量产品所耗费的支出
    2. 显性成本：（可见成本）有些生产要素在购买时需要支付报酬，一般用货币来支付
    3. 隐性成本：有些生产要素报酬不需要立即支付，或合同规定必须支付，如企业家才能
    4. 机会成本：（显性成本+隐性成本）生产要素用于目前这种用途时所放弃的其他用途中的最高收入
    5. 可变成本(VC)：短时间内随产量变化而变化的那部分生产要素报酬
    6. 固定成本(FC)：短时间内不随产量变化而变化的那部分生产要素的报酬
    7. 边际成本(MC)：额外一单位产量所引起的总成本的增加（MC = TC变化量/Q变化量）
    8. 边际成本函数和平均成本函数图形：均为“U”型

第四章

  1. GDP
  2. GNP
  3. 如何计算GDP
  4. 名义GDO
  5. 实际GDP
  6. GDP变动的两个因素
  7. GDP平减指数
  8. 名义GDP增速
  9. GDP的局限性
  • 答案：
    1. GDP：一定时期内一国（或地区）境内生产的所有最终产品于服务的市场价值总和。衡量生产而非销售的最终产品于服务的价值
    2. GNP：一个国民概念，一定时期内某国国民拥有的生产要素所生产的最终产品和劳务市场的价值
    3. 如何计算GDP：
      1. 生产法（增值法）：通过核算各行各业在一定时期中生产的价值增值来计算GDP
      2. 支出法（最终产品法）：通过核算在一定时期内整个社会购买最终产品的总支出即最终产品的总卖价来计算GDP
        • 用支出法核算GDP包括：消费(C)，投资(I)，政府购买(G)和净出口(X-M)
        • GDP = C + I + G + (X - M)
        • C消费：居民或个人用于物品和劳务的支出。C = 耐用消费品 + 非耐用消费品 + 劳务
          • 耐用消费品：汽车、电视机、洗衣机
          • 非耐用消费品：食物、衣服、水果
          • 劳务：医疗、理发、旅游
          • 注意：居民消费不包括建造住宅或购买住宅的支出
        • I投资：增加或更换资本资产的支出；I = 固定资产投资 + 居民住宅投资 + 存货投资
          • 固定资产投资：厂房、大型机械设备
          • 存货投资：存货价值
        • G政府购买：各级政府购买物品和劳务的支出，包括公务员的薪水和用于公务的支出
          • 如：政府建立法院、提供国防、外交、公共交通和公共教育等服务
          • 注意：政府够没只是政府支出的一部分
          • 政府支出 = 政府购买 + 政府转移支付 + 公债利息
          • 政府转移支付和公债利息不计入GDP中
          • 净出口：国外购买国内生产的物品与国内购买外国生产的产品之间的差额
      3. 收入法：通过核算要素收入亦即企业生产成本计量GDP
        • GDP = 工资 + 利息 + 租金 + 利润 + 间接税 + 企业转移支付 + 折旧
    4. 名义GDP：用当年价格计算的最终产品和服务的市场价值
    5. 实际GDP：用从前某一年作为基年的价格，计算的当年最终产品和服务的市场价值
    6. GDP变动的两个因素：1.所生产的物品和劳务数量的变动 2.物品和劳务价格的变动
    7. GDP平减指数：名义GDP和实际GDP的比率，反映通胀的物价指数
    8. 名义GDP增速 = 经济增长速度 + 通货膨胀率
    9. GDP的局限性：1.没有考虑非市场性活动，如：地下经济，非市场经济活动（家务劳动，自给自足生产）， 2.没有考虑环境破坏活动， 3.没有衡量效益、效率、质量和实际国民财富，没有考虑收入分配状况， 4.不是衡量福利的完美指标

第五章

  1. 失业
  2. 总人口分类
  3. 劳动人口分类
  4. 失业率
  5. 劳动力参工率
  6. 失业类型
  7. 充分就业
  8. 通货膨胀
  9. 通货膨胀的产量效应
  10. 通货膨胀的收入再分配效应
  11. 通货膨胀的财富再分配效应
  • 答案：
    1. 失业：在当前工资水平下愿意工作的人无法找到工作的一种状态。
      • 失业者3条件：年龄16-65；有工作能力；愿意工作但没有工作
    2. 总人口分为：成年人口 + 非成年人口。成年人口分为：劳动力人口 + 非劳动力人口
    3. 劳动力人口 = 就业人口 + 失业人口
    4. 失业率 = 失业人口占劳动人口的百分比
    5. 劳动力参工率：劳动力占成年人口的百分比
    6. 失业类型：摩擦性失业，结构性失业，周期性失业
      1. 摩擦性失业：经济中由于劳动力的正常流动而引起的失业。
        • 特点：经济中失业人数与职位空缺额相当。摩擦性失业时由于劳动者流动不充分及劳动力市场信息不灵等原因造成的。
        • 摩擦性失业通常起源于劳动力的供给方，是过渡性的，短期性的
      2. 结构性失业：劳动者的技能与空闲的工作岗位不匹配造成的失业。
        • 特点：失业和职位空缺大体相等，但劳动力的供求长期不一致，往往属于非自愿失业。“失业与空位”并存。通常持续时间较长
        • 结构性失业通常起源于劳动力的需求方
        • 摩擦性失业与结构性失业属于自然失业
      3. 周期性失业：总需求不足造成。
        • 一般情况下政府只有采用扩大购买的积极财政政策，才能刺激有效需求扩大就业（凯恩斯主义），伴随而来的可能是通货膨胀
    7. 充分就业：除了摩擦性失业和结构性失业外，所有愿意工作的人都在从事工作的状态
    8. 自然失业率：充分就业时的失业率。 = 摩擦性失业率 + 结构性失业率
    9. 通货膨胀：在纸币流通的制度下，在一定的时期内，由于货币发行量超过流通中实际货币需求量而引起的物品和劳务价格的普遍持续性的上涨
    10. 通货膨胀的产量效应：从长期来看，通货膨胀与产量之间不存在必然关系。一般来说，在短期内，由于预期之外的需求拉上的膨胀会使产品价格的上涨快于货币工资率的上涨，实际工资率会有所降低，从而促使企业增雇工人，扩大产量以谋取利润，使就业和国民产出增加
    11. 通货膨胀的收入再分配效应：通货膨胀使一部分人实际收入增长，另一部分人实际收入减少的经济影响。通胀有利于债务人、雇主、政府；不利于债权人、工人及公众
    12. 通货膨胀的财富再分配效应：对拥有货币财产与非货币财产者财产净值的影响。对于持有不变价格财产的人来说，其拥有的证券、银行存款的票面价值是相对固定的，其实际价值将随物价上涨而下降。对于持有可变价值财产的人来说，则正好相反，他们会因通货膨胀而受益
